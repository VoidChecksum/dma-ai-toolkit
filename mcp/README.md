# dma-ai-toolkit MCP

Read-only MCP server for source-backed DMALibrary agent workflows.

## Run

```bash
python mcp/dma_mcp.py
```

## Tools

- `overview`
- `recommend_context`
- `api_lookup`
- `validate_answer`
- `get_file`
- `list_docs`

The server does not write files, scan networks, or fetch remote targets.
